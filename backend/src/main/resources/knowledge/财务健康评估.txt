财务健康评估指南

一、财务健康的核心指标

1. 储蓄率
- 每月储蓄金额 / 每月收入 x 100%
- 健康标准：20%~30%
- 计算公式：（收入 - 支出）/ 收入

2. 负债收入比
- 月供总额 / 月收入 x 100%
- 健康标准：不超过30%
- 超过40%属于财务压力过大

3. 紧急备用金充足度
- 当前可动用资金 / 月平均支出
- 健康标准：3~6个月
- 低于3个月属于不安全状态

4. 资产配置合理性
- 是否分散在不同类型的资产中
- 是否符合个人的年龄和风险承受能力
- 是否有合理的再平衡计划

5. 保险覆盖率
- 是否配置了必要的保险
- 保额是否充足
- 保费是否在合理范围内

二、财务健康评级

A级 优秀：满足所有健康标准
- 储蓄率 >= 20%
- 负债率 <= 30%
- 备用金 >= 6个月
- 有完善的投资和保险配置

B级 良好：基本满足健康标准
- 储蓄率 10%~20%
- 负债率 30%~40%
- 备用金 3~6个月
- 有基本的投资和保险

C级 一般：部分未达标
- 储蓄率 < 10%
- 负债率 40%~50%
- 备用金 1~3个月
- 投资和保险不足

D级 需改善：多项未达标
- 储蓄率接近0
- 负债率 > 50%
- 备用金 < 1个月
- 无投资和保险

三、财务体检频率
- 全面体检：每年一次
- 常规检查：每季度一次
- 快速检查：每月核对预算执行情况

四、财务改善行动计划
1. 建立紧急备用金（最优先）
2. 优化支出结构，提高储蓄率
3. 制定清晰的还债计划
4. 建立投资计划并执行定投
5. 完善保险配置
6. 定期检视和调整
